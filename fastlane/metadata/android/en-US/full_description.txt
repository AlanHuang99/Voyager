Voyager is an open-source file manager for Android with built-in SFTP, FTP, SMB, and WebDAV access, built with Material Design 3 and Jetpack Compose.

Features:
• Local file browsing with file management (copy, move, rename, delete, create)
• SFTP remote access, with password, keyboard-interactive, or private key authentication
• FTP remote access
• SMB/CIFS network share browsing
• WebDAV remote access
• Copy and move files between local storage and a remote server, in both directions
• Download remote files and folders to the device
• Multiple browser sessions for local and remote locations, with a session switcher
• Mounted external storage volumes, such as SD cards and USB/OTG, listed alongside internal storage
• 20 built-in themes including Catppuccin, Nord, Solarized, Gruvbox, Rosé Pine, Tokyo Night, and High Contrast
• Custom theme support
• AMOLED black theme
• List and grid view modes
• Bookmarks and quick access
• Storage usage overview
• Hidden files toggle
• Multiple sort options (name, size, date, type)
• Multi-select with copy/cut/paste
• Material You dynamic color support (Android 12+)

Privacy:
• No analytics or tracking
• No internet access required for local browsing
• All network connections are user-initiated
• Fully open source (GPLv3)
